# MUTREICH – Coaching Website

**„Der Weg beginnt mit Mut."**

Offizielle Website der Coaching-Marke MUTREICH von Jan Peter „Pit" Jansen.

---

## Über das Projekt

Einseitige Marketing-Website (Single Page) für das Coaching-Angebot von MUTREICH.  
Gebaut als einzelne `index.html` – kein Framework, kein Build-Prozess, sofort einsatzbereit.

**Live:** https://jpjansen00-code.github.io/mutreich-homepage/
